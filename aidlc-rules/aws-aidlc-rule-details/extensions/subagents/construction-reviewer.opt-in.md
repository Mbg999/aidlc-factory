# Construction Reviewer — Opt‑in

¿Quieres habilitar el subagente "Construction Reviewer" para este flujo?

Si se habilita, este subagente ejecutará revisiones automáticas de estilo, búsqueda de TODOs y detección básica de secretos, y generará `aidlc-docs/construction-review.md`.

Responde `yes` para activar o `no` para omitir.

Cuando se habilite, se cargará el archivo completo `construction-reviewer.md`.
