# Construction Builder — Opt‑in

¿Quieres habilitar el subagente "Construction Builder" para este flujo?

Si se habilita, el subagente propondrá comandos de build, ejecutará comprobaciones no destructivas opcionales y generará `aidlc-docs/build-report.md`.

Responde `yes` para activar o `no` para omitir.

Cuando se habilite, se cargará el archivo completo `builder.md`.
