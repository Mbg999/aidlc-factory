# Code Reviewer — Opt‑in

Do you want to enable the Code Reviewer subagent for this workflow?

If enabled, the Code Reviewer will run during Construction and Build & Test phases and perform lightweight linting, security-scan checks, and annotation of generated code. Answer `yes` to enable or `no` to skip.

When enabled the full rule file `code-reviewer.md` will be loaded and enforced according to the configured permissions.
