# Construction Planner — Opt‑in

¿Quieres habilitar el subagente "Construction Planner" para este flujo?

Si se habilita, el subagente ejecutará análisis durante la fase de *construction* y generará un plan de construcción en `aidlc-docs/construction-plan.md`.

Responde `yes` para activar o `no` para omitir.

Cuando se habilite, se cargará el archivo completo `planner.md` y se aplicarán las reglas y permisos definidos allí.
